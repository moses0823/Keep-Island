# 動態島常駸工具（Dynamic Island Persistence Utility）

本工具可在 **非原生支援 Dynamic Island 的 iPhone** 上，讓以各種方式啟用的「動態島視圖」維持常駸顯示。  
適用於透過 **Dynamic Cow、Misaka X、Misaka26** 等方式啟用動態島的裝置。

> 原生支援 Dynamic Island 的機型可正常運作，但實際用途有限。

---

## 功能

- 讓第三方方式啟用的動態島保持常駸  
- 不修改系統行為，不注入 SpringBoard  
- 無需越獄  
- 低資源佔用，常駐後幾乎不影響效能

---

## 相容性

- iOS 15 – iOS 18  
- 支援環境：
  - Misaka
  - Misaka26
  - Dynamic Cow
  - TrollStore
  - Sideload（AltStore / Sideloadly）
- 原生 Dynamic Island 裝置可啟動本 App，但使用情境有限

---

## 安裝

### 📌 TrollStore
1. 將 `.ipa` 透過 TrollStore 安裝  
2. 打開 App 即可使用

### 📌 Sideload（AltStore／Sideloadly）
1. 使用 sideload 工具安裝 `.ipa`  
2. 啟動後完成設定即可

---

## 已知問題

- 若動態島顯示區被其他 tweak 改動，可能需要重新啟動本 App。
- 某些非預期的系統錯誤 UI（如狀態列 glitch）可能導致顯示中斷，但重新啟動即可恢復。
